# Process Procedure P-03: System Start

Bypass start sequence for pressure vessels:
1. Ensure main feed pump is active.
2. Adjust safety bypass valve V-202 to a pressure setting of 120 PSI.
3. Verify flow rate stabilizes prior to loading pressure vessel.
